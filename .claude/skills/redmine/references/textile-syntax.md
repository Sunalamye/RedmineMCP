# Textile 富文本語法

Redmine 使用 Textile 標記語言，在 `notes` 和 Wiki 中可使用。

## 基本格式

| 效果 | 語法 |
|------|------|
| **粗體** | `*粗體*` |
| _斜體_ | `_斜體_` |
| 底線 | `+底線+` |
| 刪除線 | `-刪除線-` |
| 程式碼 | `@程式碼@` |

## 標題

```textile
h1. 標題一
h2. 標題二
h3. 標題三
```

## 清單

```textile
* 無序項目
* 另一項目

# 有序項目
# 另一項目
```

## 其他

| 效果 | 語法 |
|------|------|
| 引文 | `bq. 引文內容` |
| 連結 | `"顯示文字":http://url` |
| 圖片 | `!image.png!` |
| 程式碼區塊 | `<pre>code</pre>` |

## 範例

```textile
h2. 修復說明

*問題已修復*，變更內容：

# 修改了 A 檔案
# 修改了 B 檔案

@git commit abc123@
```
